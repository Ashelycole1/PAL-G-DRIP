# Pal G Drip House - Premium Streetwear Platform

Welcome to the official repository for **Pal G Drip House**, Uganda's premier high-end streetwear destination. This platform has been meticulously redesigned to deliver a world-class, editorial luxury experience.

## ✨ Platform Features

- **Cinematic Experience**: A high-impact, automated home slideshow showcasing the latest campaign imagery.
- **Modern E-Commerce Grid**: A refined, functional collections archive with real-time category and search filtering.
- **Dual-Theme Support**: A seamless Light and Dark mode toggle with persistent user preferences.
- **Bespoke Interaction**: Custom luxury cursor design and optimized scroll-reveal animations for a premium feel.
- **Fully Responsive**: Optimized for high-fidelity viewing on everything from desktop to mobile devices.

## 🛠️ Technology Stack

- **Frontend**: Semantic HTML5, Vanilla JavaScript (ES6+).
- **Styling**: Modern CSS3 utilizing a robust CSS Variable system for theming.
- **Design System**: A custom minimalist luxury design language with a focus on high-fashion silhouettes.

## 🚀 Deployment

The site is optimized for static hosting on platforms like **Vercel**, **Netlify**, or **GitHub Pages**.

### Steps to Deploy (Vercel):
1. Push this repository to your GitHub account.
2. Sign in to [Vercel](https://vercel.com).
3. Click "Add New Project" and select this repository.
4. Click "Deploy" — the platform will automatically detect the static files.
